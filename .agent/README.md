# Agent Harness

This directory stores agent-generated project context and collaboration artifacts.

## Structure

- `working-memory/`: active notes and scratchpad context for in-progress work.
- `checkpoints/`: progress snapshots created during long-running tasks.
- `handoffs/`: summaries for passing work between agents or contributors.

## Canonical Guidance

Use `AGENTS.md` in the repository root as the source of truth for agent behavior and process.
